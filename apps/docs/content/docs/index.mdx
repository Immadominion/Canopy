# Canopy

Canopy is a developer operations platform for Solana Mobile. It gives you three things:

| Pillar | What you get |
|---|---|
| **Beta Tracks** | Wallet-gated, time-limited APK distribution — TestFlight for Solana Mobile |
| **Web3-Native Analytics** | Event tracking keyed to wallet addresses and on-chain context |
| **Release Ops** | CI/CD integration, pre-submission checks, and dApp Store submission pipeline |

## Who is this for?

Canopy is for developers building apps for the **Seeker** device by Solana Mobile and distributing
them through the **Solana dApp Store**. If you are shipping a mobile dApp, Canopy gives you the
testing, analytics, and release tooling that the ecosystem was missing.
